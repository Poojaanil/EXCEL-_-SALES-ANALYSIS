This project presents a comprehensive sales analysis using Microsoft Excel. The dataset includes order details, sales representatives, product categories, regions, units sold, pricing, and total sales. The analysis focuses on:

Sales performance by region and category
Top-selling products and high-value orders
Sales representative performance evaluation
Revenue trends and order distribution
KPI tracking through Excel dashboards
Data visualization using charts and pivot tables
Conditional formatting to highlight key metrics

The goal is to transform raw sales data into actionable business insights that support strategic decision-making and performance optimization. Excel tools such as Pivot Tables, Pivot Charts, Lookup Functions, Conditional Formatting, and Dashboard Design were used throughout the analysis.
